Excel Financial Model — Project Overview

Project Title: 3‑Statement Financial Model

Tools: Excel (Advanced), PivotTables, Power Query

Objective:  
Build an integrated financial model combining the Income Statement, Balance Sheet, and Cash Flow Statement to support forecasting and decision‑making.

Key Features:
-Dynamic assumptions sheet
-Revenue and cost drivers
-Automated financial statements
-Scenario & sensitivity analysis
-KPI dashboard (profitability, liquidity, leverage)

Skills Demonstrated:
-Financial modelling
-Forecasting
-Excel automation
-Data validation
-Financial analysis

Dataset:  
Synthetic or anonymised financial data

Outcome:  
A professional Excel model that supports budgeting, forecasting, and strategic planning.

Tableau Dashboard — Project Overview

Project Title: Sales Performance Dashboard

Tools: Tableau, Excel

Objective:  
Provide a clear visual overview of sales trends, regional performance, and product-level insights to support business decision-making.

Key Features:
-Interactive filters for region, product, and time period
-Trend lines showing monthly and quarterly sales
-Top-performing products visual
-Regional heatmap for geographic insights

Skills Demonstrated:
-Data visualisation
-Dashboard design
-Storytelling with data
-Data cleaning and preparation

Dataset:  
Synthetic or anonymised sales data

Outcome:  
A professional dashboard that helps stakeholders understand performance drivers and identify growth opportunities.

Power BI Dashboard — Project Overview

Project Title: Budget vs Actual Dashboard

Tools: Power BI, DAX, Excel

Objective:  
Provide a clear comparison between budgeted and actual financial performance, highlighting variances and key insights for management reporting.

Key Features:
-KPI cards for revenue, expenses, and net profit
-Variance analysis with conditional formatting
-Monthly trend charts
-Drill‑through pages for department-level insights

Skills Demonstrated:
-Data modelling
-DAX calculations
-Data cleaning and transformation
-Visual design and storytelling

Dataset:  
Synthetic or anonymised financial data

Outcome:  
A professional dashboard that supports decision‑making and financial performance tracking.

SQL Project — Overview

Project Title: Sales & Customer Analysis Queries

Tools: SQL (MySQL / PostgreSQL / SQL Server)

Objective:  
Use SQL to clean, transform, and analyse sales and customer data to uncover insights that support business decision-making.

Key Features:
-Data cleaning (removing duplicates, handling nulls)
-Joins (INNER, LEFT, RIGHT)
-Aggregations (SUM, AVG, COUNT)
-Grouping and filtering
-Subqueries and CTEs
-Basic window functions

Skills Demonstrated:
-Writing efficient SQL queries
-Data wrangling
-Analytical problem-solving
-Understanding relational databases

Dataset:  
Synthetic or anonymised sales/customer dataset

Outcome:  
A set of SQL scripts that demonstrate practical data analysis skills used in real business environments.
